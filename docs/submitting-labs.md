# How to Submit Work
Canvas is the official place to submit assignments for this course.

GitHub will be used for code, starter projects, examples, and project work, but pushing code to GitHub is not the same as submitting the assignment. If an assignment has a Canvas submission link, you must submit it in Canvas.

For most coding assignments, you will submit a GitHub link in Canvas.

See the Module [Getting Started->How to Submit Work](https://uah.instructure.com/courses/88040/pages/how-to-submit-work?module_item_id=2860420) for more information.
